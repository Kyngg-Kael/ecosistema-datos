# Evidencia Técnica y Científica

En esta carpeta se encuentran los Jupyter Notebooks que documentan el proceso de investigación, análisis de datos y entrenamiento de modelos que sustentan el Sistema Multipropósito.

1. **01_Modelo_Biomasa...**: Entrenamiento de Random Forest para estimar carbono.
2. **02_Analisis_Estructura...**: Procesamiento de mapas de altura de dosel (1m). (basado en el ejercicio del ingeniero Steven Mora)
3. **03_Conectividad...**: Lógica de consulta a la API de Biodiversidad.
4. **04_Validacion...**: Pruebas de concepto con datos crudos de la NASA.
